First-time setup

Most likely onboarding path

A new operator usually goes settings → watchlists → jobs → first run → dashboard review.

  1. Configure provider credentials and summary backend from Settings.
  2. Create reusable ticker groups in Watchlists.
  3. Create a manual or scheduled job in Jobs.
  4. Run it, then use the dashboard and debugger to triage warnings.
